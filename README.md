```
1. Install Figma plugin: "html.to.design"
2. Open each prototype in Chrome
3. Copy HTML → paste in Figma plugin
4. Refine and organize into components
```

**Option C: Manual Recreation** (Most control - 1-2 weeks)
```
1. Follow FIGMA_IMPLEMENTATION_GUIDE.md
2. Use DESIGN_SYSTEM.md as reference
3. Build component library from scratch
4. Create all 4 apps with full interactions
```

### For Developers:

**Using React Prototypes:**
```
1. Review interactive prototypes (already built)
2. Use Chrome DevTools to inspect
3. Extract exact CSS values
4. Build native components (SwiftUI/Compose/Flutter)
5. Reference DESIGN_SYSTEM.md for tokens
```

**Using Figma (after designer creates it):**
```
1. Access Figma file in Dev Mode
2. Select component → Copy code
3. Get exact spacing/colors
4. Export assets (@2x, @3x)
5. Build native components
```
